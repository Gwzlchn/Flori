A Structured Document Model

A document adapter must preserve the original hierarchy instead of flattening every paragraph into an anonymous string. Stable blocks make translation, search, citations, and review point back to the same source without asking downstream consumers to guess. See the evidence specification for the complete rules.

Preserved structures

Lists, code, figures, tables, footnotes, and safe embed descriptors retain their reading order and source locator. The adapter records remote assets without executing scripts or rewriting the immutable HTML input, so later download and rendering stages remain auditable.
